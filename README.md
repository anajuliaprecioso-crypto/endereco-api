# 📦 Busca de CEP com Kotlin

Projeto desenvolvido em Kotlin que realiza a busca de informações de um CEP utilizando API externa.

---

## 🚀 Funcionalidades

- 🔎 Consulta de CEP
- 📍 Retorno de dados como:
  - Logradouro
  - Bairro
  - Cidade
  - Estado
- ⚡ Consumo de API (ex: ViaCEP)
- 🧩 Código simples e organizado

---

## 🛠️ Tecnologias utilizadas

- Kotlin
- IntelliJ IDEA
- API ViaCEP
- (adicione outras, se tiver usado, como Retrofit, Ktor, etc)

---

## 📡 Como funciona

O usuário informa um CEP, e o sistema faz uma requisição para a API e retorna os dados correspondentes.
